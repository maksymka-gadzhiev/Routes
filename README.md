# Маршрутизатор: Веб-приложение для создания и управления маршрутами

Проект представляет собой Full-stack решение для работы с географическими маршрутами. Пользователи могут создавать, сохранять и делиться маршрутами, управлять друзьями и просматривать статистику.

## Технологии
**Backend**:
- Java 17
- Spring Boot 3
- Spring Security
- JWT аутентификация
- Yandex Maps API

**Frontend**:
- React 18
- React Router 6
- Yandex Maps JS API 2.1
- CSS Modules

## Основные эндпоинты API

### Аутентификация
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/auth/register` | Регистрация нового пользователя |
| POST | `/api/auth/login` | Авторизация пользователя |
| GET | `/api/auth/validate` | Проверка валидности JWT токена |


### Управление друзьями
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/friends/invite` | Отправить приглашение в друзья |
| POST | `/api/friends/accept/{inviteId}` | Принять приглашение |
| POST | `/api/friends/decline/{inviteId}` | Отклонить приглашение |
| GET | `/api/friends` | Список друзей текущего пользователя |
| GET | `/api/friends/invites` | Входящие запросы на дружбу |

### Работа с маршрутами
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/routes` | Создание нового маршрута |
| GET | `/api/routes/user/{userId}` | Получение маршрутов пользователя |
| POST | `/api/routes/export/gpx` | Экспорт маршрута в GPX |
| POST | `/api/routes/import/gpx` | Импорт маршрута из GPX |


### Профиль пользователя
| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/users/{userId}/profile` | Получение профиля пользователя |

## Фронтенд-роуты
| Путь | Компонент | Описание |
|------|-----------|----------|
| `/` | Home | Главная страница с категориями маршрутов |
| `/login` | Login | Страница авторизации |
| `/registration` | Registration | Страница регистрации |
| `/map` | Map | Интерактивная карта с созданием маршрутов |
| `/profile` | Profile | Личный профиль пользователя |
| `/saveRoute` | SaveRoute | Страница сохранения маршрута |
| `/friends` | FriendsPage | Управление друзьями |

## Основной функционал
1. **Работа с картой**:
   - Добавление/удаление меток
   - Построение маршрутов для разных типов транспорта
   - Импорт/экспорт в GPX-формате
   - Визуализация маршрутов

2. **Социальные функции**:
   - Система друзей с запросами
   - Просмотр профилей друзей
   - Настройки видимости маршрутов

3. **Статистика**:
   - Количество созданных маршрутов
   - Пройденная дистанция
   - Графическое отображение активности
